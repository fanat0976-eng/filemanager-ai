# File Manager AI

CLI файловый менеджер с AI-помощником.

## Установка

```bash
pip install textual httpx rich
```

## Запуск

```bash
python main.py
```

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| Tab | Переключить панель |
| Enter | Открыть папку/файл |
| Backspace | Вернуться в родительскую папку |
| F2 | Переименовать |
| F3 | Копировать |
| F4 | AI: переименовать умно |
| F5 | Обновить |
| F6 | AI: найти по описанию |
| F7 | AI: организовать файлы |
| F8 | AI: описание папки |
| Delete | Удалить (в корзину) |
| Ctrl+D | Найти дубликаты |
| Q | Выйти |

## AI-команды (в командной строке)

| Команда | Описание |
|---------|----------|
| `ai <текст>` | Спросить AI |
| `cd <папка>` | Перейти в папку |
| `mkdir <имя>` | Создать папку |
| `stats` | Статистика папки |
| `old <дней>` | Найти старые файлы |

## Примеры AI-запросов

```
ai переименуй файлы в читаемые имена
ai найди python файлы
ai что в этой папке?
ai организуй файлы по типу
```

## Зависимости

- Python 3.10+
- Textual (TUI)
- Rich (форматирование)
- httpx (AI запросы к Ollama)
- Ollama (локальный AI)
